# Excel-Auswertung
Excel-Auswertungs-Tool ist eine einfache Windows-Anwendung, mit der Sie Excel-Dateien automatisch auswerten und übersichtlich aufbereiten können – ganz ohne Formeln, Makros oder Excel-Vorkenntnisse.  Ideal für Selbstständige, kleine Unternehmen und Vereine, die schnell Klarheit über Zahlen brauchen.
